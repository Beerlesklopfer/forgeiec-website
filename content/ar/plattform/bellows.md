## Bellows -- بوابة OPC UA

**قيد التطوير**

Bellows هي بوابة OPC UA لمنصة ForgeIEC. منفاخ المسبك يغذي النار -- Bellows
يغذي الاتصال بين أنظمة الأتمتة والبنية التحتية لتكنولوجيا المعلومات.

---

## الاتصال من آلة إلى آلة

OPC UA (Open Platform Communications Unified Architecture) هو معيار الاتصال
للصناعة 4.0. سيوفر Bellows خادم OPC UA كاملا يكشف متغيرات PLC للأنظمة
العليا.

### حالات الاستخدام المتوقعة

- **تكامل SCADA** -- ربط وحدات PLC بأنظمة المراقبة الحالية
- **تبادل بيانات M2M** -- اتصال مباشر بين وحدات PLC وأنظمة الطرف الثالث
- **بوابة IT/OT** -- جسر بين شبكات الأتمتة والبنية التحتية لتكنولوجيا المعلومات
- **أرشفة البيانات** -- توفير بيانات العمليات للأرشفة

---

## البنية المعمارية المخططة

سيعمل Bellows كعملية مستقلة يديرها خادم `anvild`. يتم استقبال بيانات
العمليات عبر Anvil (IPC بدون نسخ) وكشفها عبر بروتوكول OPC UA.

```
PLC  --->  anvild  --->  Bellows (خادم OPC UA)  --->  عملاء OPC UA
            Anvil IPC                                  SCADA، MES، السحابة
```

### الميزات المخططة

- خادم OPC UA متوافق مع المواصفات
- كشف تلقائي لمتغيرات IEC
- نموذج معلومات قابل للتكوين
- التشفير والمصادقة
- اكتشاف تلقائي للخدمات
- سجل بيانات مدمج
