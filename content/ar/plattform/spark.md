## Spark -- نفق Zenoh

Spark هو جسر الشبكة من الحافة إلى السحابة لمنصة ForgeIEC. الشرارة تشعل
النار -- Spark يشعل الاتصال بين وحدات PLC في الموقع والخدمات السحابية.

---

## من الحافة إلى السحابة بدون تنازلات

تحتاج المنشآت الصناعية الحديثة إلى اتصال موثوق بين معدات الحقل والخدمات
السحابية -- للصيانة عن بعد وتحليل البيانات والمراقبة عن بعد. يوفر Spark
هذا الاتصال عبر بروتوكول Zenoh.

### لماذا Zenoh؟

Zenoh هو بروتوكول اتصال مصمم للبيئات المقيدة والموزعة. على عكس شبكات VPN
التقليدية أو اتصالات MQTT، يوفر Zenoh:

- **اجتياز NAT أصلي** -- بدون تكوين جدار حماية معقد
- **بروتوكول pub/sub فعال** -- استهلاك منخفض لعرض النطاق الترددي
- **توجيه تكيفي** -- اختيار تلقائي لأفضل مسار شبكة
- **أدنى زمن استجابة** -- مصمم لتطبيقات الوقت الفعلي

---

## حالات الاستخدام

### الصيانة عن بعد

اتصال آمن بوحدات PLC البعيدة للتشخيص وتحديث البرامج وقراءة المتغيرات --
بدون الحاجة للسفر إلى الموقع.

### جمع البيانات السحابية

نقل بيانات العمليات إلى منصات سحابية (AWS، Azure، بنية تحتية خاصة)
للتحليل والتعلم الآلي والصيانة التنبؤية.

### مراقبة متعددة المواقع

مراقبة مركزية لعدة منشآت من نقطة واحدة، مع بيانات في الوقت الفعلي وأدنى
زمن استجابة.

---

## البنية المعمارية

يعمل Spark كخادم على وحدة PLC، متصل بوقت التشغيل عبر Anvil (IPC بدون نسخ).
يتم نقل البيانات بشكل انتقائي إلى عقد Zenoh البعيدة.

```
الموقع أ                         السحابة / الموقع المركزي
+------------+                   +------------------+
| anvild     |                   | موجه Zenoh       |
|   |        |                   |   |              |
|   +- Spark |----- Zenoh ---------|  +- خدمات      |
|   |  Anvil |   (مشفر)          |     تحليلات     |
+------------+                   +------------------+
```

### الخصائص

- تشفير من طرف إلى طرف (TLS 1.3)
- تصفية قابلة للتكوين للمتغيرات المنقولة
- إعادة اتصال تلقائية عند انقطاع الشبكة
- ضغط البيانات للروابط منخفضة النطاق الترددي
- متوافق مع الشبكات المحمولة (4G/5G)

---

## التفاصيل التقنية

- **البروتوكول**: Zenoh (zero overhead network protocol)
- **النقل**: TCP، UDP، WebSocket
- **التشفير**: TLS 1.3
- **المنصات**: x86_64، ARM64، ARMv7 (Linux)
- **التكامل**: Anvil IPC نحو وقت التشغيل، Zenoh نحو السحابة
