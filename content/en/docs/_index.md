Welcome to the ForgeIEC documentation. ForgeIEC is an open-source IEC 61131-3
PLC editor and runtime for Linux.

## Components

| Package | Description |
|---------|-------------|
| **forgeiec** | Qt6 Editor + iec2c compiler (amd64, arm64) |
| **forgeiecd** | Rust gRPC runtime server (amd64, arm64, armhf) |
| **forgeiec-examples** | Example projects |
| **forgeiec-doc** | This documentation |
