Partenaires

Partenariat chez ForgeIEC

ForgeIEC est un projet open source porte par la conviction que l’automatisation industrielle doit etre accessible a tous. Nous recherchons des partenaires qui partagent cette vision et sont prets a faconner activement l’avenir de l’automatisation.

Conditions prealables

Engagement open source demontre

Les candidats doivent avoir deja prouve leur soutien a la philosophie open source. Cela signifie :

  • Contributions verifiables a des projets open source existants (commits, pull requests, rapports de bugs, documentation)
  • Profil public sur des plateformes telles que GitHub, GitLab ou Codeberg
  • Participation active aux communautes open source — pas seulement l’utilisation, mais la contribution

Nous croyons que les actes parlent plus fort que les mots. Ceux qui considerent l’open source uniquement comme un outil marketing sans contribuer eux-memes ne correspondent pas a notre philosophie.

Valeurs partagees

  • Transparence — Communication ouverte, decisions tracables
  • Collaboration — Revues de code, discussions architecturales communes
  • Qualite — Tests documentes, code propre, fiabilite industrielle
  • Engagement a long terme — Le partenariat est un marathon, pas un sprint

Ce que nous offrons

Acces a la plateforme ForgeIEC

  • Acces anticipe aux nouvelles fonctionnalites
  • Ligne directe avec l’equipe principale
  • Possibilite de faconner activement la feuille de route

Developpement conjoint

  • Travail collaboratif sur les pilotes de bus industriels
  • Integration de materiel specifique aux fabricants
  • Developpement d’extensions sectorielles

Support technique

  • Correction de bugs prioritaire
  • Conseil en architecture pour les extensions personnalisees
  • Support pour l’integration dans les systemes existants

Candidature

Vous souhaitez devenir partenaire ? Ecrivez-nous avec :

  • Description de votre organisation
  • Liens vers vos contributions open source
  • Votre motivation pour un partenariat
  • Domaine concret ou vous souhaitez contribuer

Ensemble, nous forgeons les outils de l’avenir.
